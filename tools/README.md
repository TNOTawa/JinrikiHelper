## 安装步骤

1. 下载 `mfa_engine_win64.zip`
2. 解压到本目录，确保目录结构如下：

```
tools/
└── mfa_engine/
    ├── python.exe
    ├── Scripts/
    │   └── mfa.exe
    ├── Lib/
    └── ...
```

3. 验证安装：运行程序后在「模型下载」页面检查 MFA 状态

## 注意事项

- MFA 引擎仅支持 Windows 64位系统
- 解压后约占用 500MB 磁盘空间
- 首次运行可能需要较长时间初始化
